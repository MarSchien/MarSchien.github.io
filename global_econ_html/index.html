Summary: Visualizing Global Economics

This webpage showcases a project from the Berkeley data science bootcamp. The goal of the project was to visually display economic data that would elucidate which regions of the world have the highest economic impact and growth. The data was retrieved from a United Nations statistics and population division, that compiled data the UN received from twenty international statistical sources represented by global regions. All data (except for GDP growth, 2005) are based on the year 2017.

The UN stored various economic information from all countries of the world in a csv file. This file was read into a jupyter notebook, where the data could then be organized into pandas dataframes using the python language. After cleaning and aggregating the data, the information was visualized using matplotlib functionality. This webpage was created using bootstrap, following the creation of an HTML skeleton and CSS styling. Please explore this webpage!
